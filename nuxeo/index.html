Welcome to the Nuxeo Platform

This wizard will help you configure your Nuxeo application in a few quick steps so you can leverage all the features of the Nuxeo Platform.

When these steps are completed, the Nuxeo server will restart automatically with your settings. You won't need to configure your application again.

If you want to use the default test configuration, just leave the default parameters at each step. You can also change these settings later, in the Admin Center.
